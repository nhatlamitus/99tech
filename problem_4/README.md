# Problem 4: Three Ways to Sum to N

## How to Run

### Prerequisites
- Node.js

### Running the Program

1. **Change file mode:**
   ```bash
   chmod +x run.sh
   ```

2. **Run command:**
   ```bash
   ./run.sh
   ```

### Expected Output
```
Sum to 1000:
Iterative: 500500
Formula: 500500
Recursive: 500500
```
