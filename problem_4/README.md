# Problem 4: Three Ways to Sum to N

## How to Run

### Prerequisites
- NodeJS

### Running the Program

1. **Install ts-node**
   ```bash
   npm install -g ts-node
   ```

2. **Run command:**
   ```bash
   ts-node main.ts
   ```

### Expected Output
```
Sum to 1000:
Iterative: 500500
Formula: 500500
Recursive: 500500
```
